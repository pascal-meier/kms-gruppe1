# To-Do App

Eine einfache To-Do Anwendung zur Verwaltung von Aufgaben und Prioritäten.

## Technologien

* Vue 3
* TypeScript
* Vite
* Vitest (Tests)

---

# Voraussetzungen

Folgende Software muss installiert sein:

* Node.js (Version 20 oder neuer)
* npm

Überprüfen mit:

```
node -v
npm -v
```

---

# Projekt starten

Repository klonen und in den Projektordner wechseln:

```
cd kill-my-self
```

Abhängigkeiten installieren:

```
npm install
```

Entwicklungsserver starten:

```
npm run dev
```

Die Anwendung ist danach erreichbar unter:

```
http://localhost:5173
```
